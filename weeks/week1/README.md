# Week 1 — Data Lab（位运算/整数/浮点）

目标：用 **纯位运算 + 推理** 建立对整数表示/溢出/浮点编码的可操作直觉。

本周你每天 90 分钟，建议节奏：
- 30m 读/看例子
- 50m 写题/写 demo
- 10m 复盘记录

## 今日任务模板（每天通用）
- 先跑：`make test`
- 做题：在 `solutions/` 里写
- 记录：在 `notes.md` 追加 3 行（今天做了什么/卡点/结论）

## 本周交付物（验收）
- `demos/` 下至少 3 个可运行 demo
- `solutions/` 下至少 10 道 bit 题（含解释）
- `notes.md`：每天至少 3 行总结

## 建议阅读（对应 CSAPP 第 2 章）

- 移位、位运算
- 溢出与边界
- IEEE 754 浮点
