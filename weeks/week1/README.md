## Week1 总结报告

### 2) 常用位操作
- `~x`：按位取反。
- `x & y`：按位与，常用于掩码保留。
- `x | y`：按位或，常用于置位。
- `x ^ y`：按位异或，不同为 1。
- `x << k`：左移，低位补 0。
- `x >> k`：右移；`int` 常见算术右移，`uint` 为逻辑右移。
- `!x`：逻辑非，结果只会是 `0/1`。
- `!!x`：把任意非零值规范成 `1`。

### 3) 整数类型表达（int32/uint32）
- `int32`（补码）范围：`[-2^31, 2^31-1]`。
- `uint32` 范围：`[0, 2^32-1]`。
- 关键边界：`Tmax=0x7fffffff`，`Tmin=0x80000000`。
- 补码恒等式：`-x = ~x + 1`。

### 4) 溢出与比较规则
- 有符号加法溢出条件：同号相加且结果异号。
- 无符号加法按 `mod 2^32` 回绕。
- 比较题（如 `x<=y`）应先分异号/同号，避免直接做差导致误判。

### 5) 浮点（float32）表达
- 结构：`sign(1) + exp(8) + frac(23)`。
- 规格化数：`(-1)^sign * (1 + frac/2^23) * 2^(exp-127)`。
- 非规格化数：`exp=0, frac!=0`，用于填补最小规格化数与 0 之间的区间。
- 特殊值：`exp=255, frac=0` 为 `Inf`，`exp=255, frac!=0` 为 `NaN`。
- 十进制小数（如 `0.1`）常无法二进制有限表示，因此存储为近似值。

### 6) Week1 能力产出
- 能从十六进制/二进制输出反推整数与浮点语义。
- 能写出 Data Lab 风格位运算解法并解释正确性。
- 能对边界值（`Tmax/Tmin/-1/0/Inf/NaN`）做系统性验证与复盘。
