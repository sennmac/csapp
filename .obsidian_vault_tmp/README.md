# Programmer Obsidian Vault

这个 vault 面向程序员的日常工作流，重点是：

- 用 `01 Daily` 记录当天进度、问题和下一步
- 用 `02 Projects` 维护项目背景、命令、排障记录
- 用 `04 Notes` 沉淀技术知识和源码理解
- 用 `05 Snippets` 保存可复用命令、SQL、脚本和代码片段

## 目录说明

- `00 Inbox`：随手记录，之后每周整理
- `01 Daily`：每日工作日志
- `02 Projects`：按项目拆分的长期笔记
- `03 Areas`：持续关注的领域，如 Backend、DevOps、Database
- `04 Notes`：结构化知识笔记
- `05 Snippets`：命令和代码片段
- `06 Archive`：结束项目和旧资料
- `07 Templates`：Obsidian 模板

## 推荐用法

1. 每天先开一页 `01 Daily`
2. 每个正式项目新建一页项目笔记
3. 遇到问题按“问题 / 原因 / 排查 / 解决 / 结论”记录
4. 每周整理一次 `00 Inbox`
5. 高频命令优先沉淀到 `05 Snippets`
